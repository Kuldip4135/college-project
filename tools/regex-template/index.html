Username Validation[2-10 Character Long Username]

/^[a-zA-Z]([0-9a-zA-Z]){2,10}$/

Password Validation

(?=.*\d)(?=.*[a-z])(?=.*[A-Z])(?!.*\s)(?=.*[!@#$*])

Email Validation

/^[^ ]+@[^ ]+\.[a-z]{2,3}/

Mastercard Validation

^(?:5[1–5][0–9]{2}|222[1–9]|22[3–9][0–9]|2[3–6][0–9]{2}|27[01][0–9]|2720)[0–9]{12}$

VisaCard Validation

^4[0–9]{12}(?:[0–9]{3})?$

CSV Validation

(?:\s*(?:\"([^\"]*)\"|([^,]+))\s*,?)+?

Phone Number Validation

((\+*)((0[ -]+)*|(91 )*)(\d{12}+|\d{10}+))|\d{5}([- ]*)\d{6}
